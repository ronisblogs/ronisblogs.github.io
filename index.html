Other Improvements

 

  • HTML5 throughout - all your pages updated to the latest HTML5 standards as soon as you open your project.  If you use non-standard custom or 3rd party code, you may need to correct it, but everything else is taken care of for you automatically.
  • Shadow, Text Shadow & letter spacing.
  • Form email templates support span, list tags and CSS styles.
  • Photo Gallery supports transparent background.
  • Fixed potential hang renaming in the Files palette (rare).
  • Improved duplicate to include leading digits in filename e.g. page001 > page 002
  • Fixed some default padding/border values being ignored.
  • Enhanced YouTube Video object.
  • Supports static (always open) menus.
  • Visual paper, snap to paper edges.
  • Spellcheck uses system dictionary on Windows 8+.
  • New text editor with webfont support, can now apply formatting before/as you type.
  • Text strikethrough support.
  • Page Include has a 'position fixed' option to lock objects to the browser window, so they don't scroll.
  • New Core folder for system items, such as freeform and webfonts.
  • New Mobile Redirect object directs small mobile devices to an alternative mini-site automatically.

Menus

Audio & Video Players

 

New audio & video player objects using the latest media standards with an option for twin media formats.  Poster image, mute and reflection for browsers that support it. 

 

Video

Supports mp4 (h264), ogg and webm.

Recommend mp4 and webm video used together for full cross-browser coverage.

 

Audio

Supports aac, mp3, ogg, opus, webm, wav.  Recommend aac and ogg audio used together for full cross-browser coverage.

 

Translucent Colours

 

Colour mixer gains new Alpha channel support.

 

You can now mix translucent colours for the background of an object, without affecting its text.

SVG Vector Graphics

 

Import vector graphics - unlike jpeg and png, svg images can be resized without pixelating.  Perfect for charts and diagrams.

 

You can create professional svg images using free software such as Inkscape - just make certain you save an optimised svg, as the default svg output is not suitable for web use.
